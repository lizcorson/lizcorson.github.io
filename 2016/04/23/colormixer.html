Color Mixer

Apr 23, 2016

In May I’ll be running a STEM workshop for Daisy and Brownie Girl Scouts. One of the activities is to teach them about the electromagnetic spectrum. We have a UV light demo using a blacklight and UV-sensitive beads, but I also wanted to add a more interactive component for visible light.

I made a color mixer using a traditional RGB LED, with one pin per color channel, as well as a WS2812B LED. The student can turn each of three potentiometers to control the input of each of the R, G, and B channels. The goal is to demonstrate additive color mixing, since most kids are only familiar with subtractive color mixing as seen in paint.

I hope to make some sort of case to neaten this up, but I may not have time. The code and a bill of materials are available in my GitHub repository here.
